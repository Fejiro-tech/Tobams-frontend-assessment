# Tobams Group Website

## Overview

A responsive website built with **Next.js** and **Tailwind CSS** for the Tobams Group Frontend Assessment. It showcases learning programs, training sections, testimonials, and company information using a modern component-based architecture, designed from a Figma UI.

---

## Live Demo
```
[View Live Project](https://tobamsfrontendassessment.vercel.app/)

```

---

## Tech Stack

- **Next.js** (App Router)
- **React**
- **Tailwind CSS**
- **Next/Image** for optimized images

---

## Project Structure

The project follows a component-based architecture where each section of the homepage is broken into reusable components for better maintainability and scalability.
---

## Setup Instructions

### 1. Clone the repository

```bash
git clone https://github.com/Fejiro-tech/Tobams-frontend-assessment.git

```

### 2. Install dependencies

```bash
npm install
```

### 3. Run the development server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to view the project.

- Fully responsive across mobile, tablet, and desktop
